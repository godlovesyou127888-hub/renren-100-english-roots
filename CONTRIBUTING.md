# 貢獻指南

**結論：所有貢獻都要幫助學員更容易學會，並幫助 Skywalker 更快測試市場。**

歡迎共同編輯教材、補充字根、設計小測驗、撰寫社群貼文與短影音腳本。

## 寫作語氣

- 使用繁體中文。
- 先講結論，再講例子。
- 適合台灣學習者。
- 對樂齡學員要溫暖、清楚、極白話。
- 不嘲笑英文不好的人。
- 不用過度學術化的語言。

## 教材原則

每個字根建議包含：

- 字根本身。
- 中文意思。
- 3 個常見單字。
- 一句白話解釋。
- 一題小測驗。
- 一句可用於招生或鼓勵學習的話。

## 樂齡友善檢查

提交前請確認：

- 句子不要太長。
- 不要一次塞太多單字。
- 不要只講字源，要講生活用途。
- 字根解釋要能被沒有英文基礎的人理解。
- 小測驗不要設計成挫折測驗。

## 避免過度學術化

可以提到字源，但不要讓課程變成語言學論文。

請常問自己：

**這對 Skywalker Co., Ltd. 目前的營收目標有直接幫助嗎？**

如果答案不明確，請先把內容縮短，留到進階補充。

## 檔案格式

- Markdown 使用 `.md`。
- 字根資料使用 CSV。
- 每日課程請參考 [lessons/day-template.md](lessons/day-template.md)。
- 價格與方案請集中放在 [marketing/pricing-and-offer.md](marketing/pricing-and-offer.md)。

## 建議流程

1. 先新增或修改一小段內容。
2. 檢查連結與 CSV 格式。
3. 寫清楚這次修改對學員或招生有什麼幫助。
4. 再提交變更。
