# 人人100英文字根計畫：10天英文開竅體驗營

結論：這 10 天內容可以直接拿去發 Line、錄短影音、做講義、測市場。

## Day 1：bio = 生命

字根意思：`bio` = 生命、人生、生物

單字例子：

1. **biology** = 生物學  
   `bio` 生命 + `logy` 學問

2. **biography** = 傳記  
   `bio` 人生 + `graphy` 書寫

3. **autobiography** = 自傳  
   `auto` 自己 + `bio` 人生 + `graphy` 書寫

一句話記憶：看到 `bio`，先想到生命、人生、生物。

小測驗：**biography** 是什麼意思？

A. 生物學  
B. 傳記  
C. 電話  
D. 望遠鏡

答案：B. 傳記

## Day 2：tele = 遠

字根意思：`tele` = 遠、遠距離

2. **television** = 電視  
   `tele` 遠 + `vision` 看

3. **telescope** = 望遠鏡  
   `tele` 遠 + `scope` 看

一句話記憶：看到 `tele`，先想到遠方、遠距離。

小測驗：**telescope** 是什麼意思？

A. 電話  
B. 電視  
C. 望遠鏡  
D. 生物學

答案：C. 望遠鏡

3. **photosynthesis** = 光合作用

## Day 5：port = 攜帶、運送

字根意思：`port` = 攜帶、運送

單字例子：

   `im/in` 進來 + `port` 運送

2. **export** = 出口  
   `ex` 出去 + `port` 運送

3. **transport** = 運輸  
   `trans` 穿越、跨過 + `port` 運送

一句話記憶：看到 `port`，先想到搬運、運送、帶來帶去。

A. 進口  
B. 出口  
C. 電視

## Day 6：spect = 看

字根意思：`spect` = 看、觀察

單字例子：

1. **inspect** = 檢查  
   `in` 進去 + `spect` 看

2. **respect** = 尊重  
   `re` 再一次 + `spect` 看  
   可以理解成：重新好好看待一個人

3. **spectator** = 觀眾  
   `spect` 看 + `ator` 的人

一句話記憶：看到 `spect`，先想到看、觀察。

小測驗：**spectator** 是什麼意思？

B. 攝影  
C. 出口  
D. 汽車

答案：A. 觀眾

## Day 7：scrib / script = 寫

字根意思：`scrib` / `script` = 寫

單字例子：

1. **describe** = 描述  
   `de` 下來 + `scribe` 寫  
   把看到的寫下來

2. **script** = 劇本、稿子  
   寫好的文字內容

3. **prescription** = 處方箋  
   `pre` 預先 + `script` 寫  
   醫師預先寫好的用藥指示

一句話記憶：看到 `scrib` 或 `script`，先想到寫、文字、稿子。

小測驗：**script** 是什麼意思？

A. 望遠鏡  
B. 劇本、稿子  
C. 電話  
D. 生物學

答案：B. 劇本、稿子

## Day 8：dict = 說

字根意思：`dict` = 說、講

單字例子：

2. **predict** = 預測

3. **dictation** = 聽寫  
   老師說，學生寫

一句話記憶：看到 `dict`，先想到說、講、預言、字典。

小測驗：**predict** 是什麼意思？

A. 預測  
B. 描述  
C. 攝影  
D. 運輸

答案：A. 預測

## Day 9：form = 形狀、形成

一句話記憶：看到 `form`，先想到形狀、形式、形成。

小測驗：**transform** 是什麼意思？

D. 聽寫

## Day 10：vis / vid = 看

字根意思：`vis` / `vid` = 看

單字例子：

1. **vision** = 視野、願景  
   跟「看見」有關

2. **video** = 影片  
   可以看的影像

3. **visible** = 看得見的  
   `vis` 看 + `ible` 可以的

一句話記憶：看到 `vis` 或 `vid`，先想到看、影像、視覺。

小測驗：**visible** 是什麼意思？

A. 看得見的

C. 進口  
D. 傳記

答案：A. 看得見的
