# 人人100英文字根計畫

結論：這是一個免費、公開、可複製的英文字根入門教材專案。

本專案由 Skywalker Co., Ltd. 發起，目標是用 100 個常見英文字根，幫助學生、家長、成人學習者與樂齡朋友，用更輕鬆的方法看懂英文單字。

## 專案定位

不用死背單字，先看懂單字裡的小零件。

英文字根就像積木。認得積木，就比較容易猜出整個單字的方向。

## 第一階段 MVP

目前先公開 10 天體驗版：

1. Day 1：bio = 生命
2. Day 2：tele = 遠
3. Day 3：auto = 自己
4. Day 4：photo = 光
5. Day 5：port = 攜帶、運送
6. Day 6：spect = 看
7. Day 7：scrib / script = 寫
8. Day 8：dict = 說
9. Day 9：form = 形狀、形成
10. Day 10：vis / vid = 看

完整內容請見：[10 天課程內容](lessons/10-day-challenge.md)
