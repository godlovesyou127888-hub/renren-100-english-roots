# 人人100英文字根計畫

**結論：這是一個免費公開教材，也是一個可以立刻測試招生與收費的 O2O 英文產品原型。**

「人人100英文字根計畫」由 Skywalker Co., Ltd. 發起，先用 10 天 MVP 讓學員體驗「字根拆解英文」的成就感，再逐步擴充為 100 個常見字根、樂齡版、親子版與付費完整班。

本專案適合：

- 想用字根快速累積英文單字的台灣學習者。
- 想重新建立英文信心的樂齡朋友。
- 想陪孩子一起學英文的家長。
- 想測試英文課程市場反應的教育創業者。

## 30 秒看懂

- **免費教材**：公開 10 天字根挑戰與未來 100 字根資料。
- **產品原型**：可直接拿去做 Line 社群、短影音、體驗班與實體課。
- **商業測試**：先用 NT$499、NT$999、NT$1,980 三種方案測需求，不等完美才推出。
- **後續擴充**：從 MVP 擴到 30 天、100 字根、樂齡版、親子版與 O2O 完整班。

## 10 天 MVP 字根

1. `bio` = 生命
2. `tele` = 遠
3. `auto` = 自己
4. `photo` = 光
5. `port` = 攜帶、運送
6. `spect` = 看
7. `scrib / script` = 寫
8. `dict` = 說
9. `form` = 形狀、形成
10. `vis / vid` = 看見

## 產品化方向

這不是只做一份教材，而是把 Gary 的英語教學、樂齡教育、AI 應用與品格教育整合成可複製的課程產品。

建議先測三條路：

1. **Line 社群 10 天挑戰**：低成本測內容吸引力。
2. **NT$499 付費體驗班**：測學員願不願意付費。
3. **NT$1,980 小班完整班**：測可否變成穩定現金流。

## 給 Skywalker 的商業提醒

先上架測試市場反應，後續再優化。

這個專案目前最重要的不是把 100 個字根寫到完美，而是先確認：

- 誰願意報名？
- 哪個族群反應最好？
- 哪個價格有人願意付？
- 哪種交付方式最省力且可複製？

如果某個細節不能直接幫助招生、收款或提高完課率，就先放到第二階段。
